## ğŸ‘¥ Step 3: Viewing All Users

Click **"All Users"** in the top navigation bar.

### The User List Table

You'll see a table with columns:
- **Roll Number**: The user's unique roll number
- **Name**: User's full name
- **Email**: User's email address
- **Mobile**: User's phone number
- **State**: User's state
- **Submission Date**: When they registered
- **Status**: Whether email was sent (green badge) or pending (yellow badge)
- **Actions**: Buttons to view details or download admit card

### Sorting
- Users are automatically sorted by newest first
- The most recent registrations appear at the top

## ğŸ” Step 4: Searching for Users

### Simple Search

1. **Find the Search Box** at the top of the user list page
2. **Type** any of the following:
   - User's email address
   - Roll number
   - Mobile number
   - Name
3. **Click "Search"** or press Enter
4. **Results** will show only matching users

### Using Filters

Below the search box, you'll find filter options:

#### Filter by State
- Click the **"State"** dropdown
- Select a state from the list
- The table will show only users from that state

#### Filter by Email Status
- Click the **"Email Status"** dropdown
- Choose:
  - **"Email Sent"** - Users who received their email
  - **"Pending"** - Users waiting for email
- The table updates automatically

#### Filter by Date Range
- **From Date**: Select the start date
- **To Date**: Select the end date
- Shows only users who registered between these dates

### Clearing Filters

Click the **"Clear"** button to remove all filters and see all users again.

## â“ Common Questions

### Q: How do I find a specific user?
**A**: Use the search box at the top. You can search by email, roll number, mobile, or name.

### Q: Can I see users from a specific state?
**A**: Yes! Use the "State" filter dropdown to select a state.

### Q: How do I know if an email was sent?
**A**: Check the "Status" column in the user list. Green badge = sent, Yellow badge = pending.

### Q: What if I can't find a user?
**A**: 
- Try different search terms
- Clear all filters
- Check if you're on the right page
- Contact your administrator

### Q: Can I edit user information?
**A**: Currently, you can only view and download. Editing features may be added in the future.

### Q: How do I log out?
**A**: Click the "Logout" button in the top right corner.

### Q: What if the page doesn't load?
**A**: 
- Check your internet connection
- Try refreshing the page
- Clear your browser cache
- Contact your administrator
